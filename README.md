# RobotLIB

Installations Instructions:

```
$ make
$ sudo make install
```

In respect to Vrep:

Is for Vrep 3.3.0, in case that you have other version change the ./RobotVREP/{remoteApi,include} to the correct version files (that you can find in VREPFolder/programing/{remoteApi,include} ).

See more on Vrep cpp Api.
